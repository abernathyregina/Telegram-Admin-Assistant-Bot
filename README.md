The **Telegram Admin Assistant Bot** is an intelligent automation system designed to simplify Telegram group administration. It automates repetitive tasks such as approving new members, filtering spam, issuing warnings, and logging user activities.

- Welcomes new members and assigns predefined roles  
- Provides command-based admin utilities (e.g., `/mute`, `/warn`, `/kick`)  
- Logs all group activities for transparency  
- Ensures consistent enforcement of group rules
